📌 Task 1: Personal Portfolio Website

🎯 Objective:
Create a personal portfolio website to showcase my profile using HTML and CSS.

🛠️ Technologies Used:

- HTML5
- CSS3
- Font Awesome Icons
- Google Fonts

📂 Project Type:
Single Page Website

📑 Sections Included:

- Header (Name & Navigation Menu)
- Home (Hero Section with introduction)
- About Me
- Skills
- Contact Form
- Footer

✨ Features:

- Fully responsive design
- Modern UI with glassmorphism effects
- Smooth navigation between sections
- Animated buttons and hover effects
- Social media links (GitHub & LinkedIn)
- Contact form with validation
- Clean layout with proper spacing and alignment

📸 Output:
A professional personal portfolio website showcasing skills, projects, and contact information in a modern UI.
